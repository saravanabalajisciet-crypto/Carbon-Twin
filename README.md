## 🏆 What is Carbon Twin AI?

**Carbon Twin AI** is a production-ready, full-stack web application that helps individuals **understand, track, and reduce their personal carbon footprint** through the power of AI. Built for **Prompt Wars**, this project demonstrates how a single, well-crafted AI prompt can generate a complete, deployable climate-tech platform.

By answering 5 simple questions about your lifestyle, Carbon Twin AI calculates your annual CO₂ emissions, generates personalised reduction strategies using **Google Gemini AI**, and lets you simulate your environmental future with interactive tools.

🌐 **Live:** [https://carbon-twin-xi.vercel.app](https://carbon-twin-xi.vercel.app/)

## 🧠 How the Carbon Calculator Works

Emission factors are derived from **IPCC** and **IEA** datasets:

```
Transport  →  Bicycle: 0.01t | Walking: 0t | Bus: 0.9t | Train: 0.3t
              Car: 2.4t | EV: 0.6t  (tonnes CO₂/year)

Food       →  Vegetarian: 1.0t | Mixed: 2.0t | Heavy Meat: 3.3t

Electricity → kWh/month × 0.00045 × 12  (avg grid carbon intensity)

Shopping   →  Low: 0.3t | Medium: 1.0t | High: 2.5t

Waste      →  Low: 0.2t | Medium: 0.6t | High: 1.2t

Score      →  100 - (total_tonnes / 8 × 100)   [0–100 scale]
```

### Sustainability Badges
| Score | Badge | Category |
|---|---|---|
| 90–100 | 🌍 | Planet Guardian |
| 70–89 | 🌱 | Eco Warrior |
| 50–69 | 🍃 | Green Learner |
| 0–49 | 🔥 | Carbon Explorer |

## 🤖 AI Integration (Gemini)

Carbon Twin AI sends a structured prompt to **Gemini 1.5 Flash** with the user's full emission profile and requests:

- Personalised summary of their carbon situation
- 5 prioritised recommendations (high / medium / low)
- Estimated CO₂ saving per recommendation (tonnes/year)
- Improved sustainability score after implementing changes

> **No API key?** No problem. The app includes a production-quality **rule-based fallback** that generates intelligent, habit-specific recommendations without any API calls.

## 🔮 Future Simulator — The Main Differentiator

The simulator is what sets Carbon Twin AI apart. Users can:

1. **Drag 4 interactive sliders** — car usage, electricity, waste, food habits
2. **See real-time projections** — current vs improved emissions update instantly
3. **View environmental equivalences** — trees planted, flights avoided, km not driven
4. **10-year area chart** — trajectory toward the Paris 2°C target

## ⚡ Getting Started Locally

```bash
# 1. Clone the repo
git clone https://github.com/saravanabalajisciet-crypto/Carbon-Twin.git
cd Carbon-Twin

# 2. Install dependencies
npm install

# 3. (Optional) Add Gemini API key
cp .env.example .env
# Edit .env → VITE_GEMINI_API_KEY=your_key_here

# 4. Start dev server
npm run dev
# → http://localhost:5173
```

### Build for Production
```bash
npm run build    # outputs to /dist
npm run preview  # preview production build locally
```
